Les balises principales

Balises Définitions
<!--...--> Définit un commentaire.
<!DOCTYPE> Définit le type de document.
<a> Définit un hyperliens.
<base> Déclare un chemin absolu qui sera utilisé comme référence pour toutes les autres url présentes dans le document.
<body> Définit le corp du document.
<br> Inserer un retour a la ligne.
<div> Divise la page en section.
<h1> to <h6> Balises de titre de niveau 1 à 6.
<head> Définit les informations à propos du document.
<hr> Trace une ligne le long de la page.
<html> Début du documents html.
<link> Définit la relation entre un lien et une ressource.
<meta> Définir des métadonnées.
<noscript> Défini une section de code HTML qui doit être insérée si l'un des scripts de la page n'est pas pris en charge ou encore si les scripts ont été désactivés par défaut dans le navigateur.
<p> Définit un paragraphe.
<script> Inserer un script.
<span> Changer le style d'un bout de texte.
<title> Définit un titre au document(dans le head).

Les balises sémantiques

Balises Définitions
<abbr> Abbréviation ou acronyme.
<address> Adresse webmaster.
<article> Définit un article.
<aside> Bloque de contenu sur le coté de la page.
<blockquote> Citation.
<cite> Définit un titre pour une oeuvre.
<details> Définit des détails supplémentaires que l'utilisateur peut afficher ou masquer.
<dialog> Boite de dialogue.
<dfn> Définition.
<footer> Section en pied de page.
<header> Section d'en-tete de la page.
<main> Contenu principal d'un document.
<nav> Barre de navigation.
<q> Citation courte.
<s> Définit un texte correcte.
<section> Creer une section dans la page.
<small> Petit texte.
<summary> Intitulé de la balise "détails".
<time> Définit une date et une heure.

Les balises de listes

Balises Définitions
<dd> Définit un description dans une liste de description.
<dl> Liste de description.
<dt> Terme a decrire dans une liste de description.
<li> Elements de liste.
<menu> Liste/menu de commande.
<menuitem> Définit un élément de commande/Menu que l'utilisateur peut appeler à partir d'un menu contextuel.
<ol> Début de liste ordonnée.
<ul> Début de liste non ordonnée.

Les balises de tableaux

Balises Définitions
<caption> Légende de tableaux.
<col> Spécifie les propriétés de chaque colonne dans un élément "colgroup".
<colgroup> Permet de faire un groupe de colonne d'un tableau pour leurs assigner les mêmes propriétées.
<table> Début d'un tableau.
<tbody> Corp du tableau.
<td> Creer une cellule dans le tableau.
<tfoot> Pied du tableau.
<th> Creer un cellule dans l'en-tete du tableau.
<thead> En-tete du tableau.
<tr> Ajoute une ligne au tableau.

Les balises de formulaires

Balises Définitions
<button> Boutton cliquable.
<datalist> Liste d'option prédéfinit pour la balise "input".
<fieldset> Encadrer plusieurs éléments de formulaire.
<form> Définit un formulaire HTML.
<input> Définit les controles utilisé dans un formulaire.
<keygen> Champ de générateur de paire de clés.
<label> Etiquettes pour la balise "input".
<legend> Légende pour la balise "fieldset".
<optgroup> Définit un groupe d'options associés dans une liste déroulante.
<option> Options des listes déroulantes.
<output> Définit le resultat d'un calcule.
<select> Liste deroulante.
<textarea> Définit un contrôle d'entrée multi-lignes (zone de texte).

Les balises de médias

Balises Définitions
<area> Définit une zone a l'interieur d'une image-map.
<audio> Contenu audio.
<canvas> Utilisé pour dessiner des graphiques, à la volée, par l'intermédiaire de scripts (habituellement JavaScript).
<figcaption> Légende pour la balise "figure".
<figure> Contenu autonome.
<img> Inserer une image.
<source> Définit des alternatives de format aux différents medias.
<track> Inserer du texte dans les medias. (sous-titres, titres, ect...).
<video> Inserer une video.

Les balises de styles

Balises Définitions
<b> Texte en gras.(Déconseillé)
<del> Texte qui a été supprimé du document mais que vous voulais garder visible.
<em> Mots/texte important(moin important que la balise "strong").
<ins> Texte insérer dans le document.
<mark> Surlignée.
<style> Inserer du style dans le documents html(Déconseillé.)
<strong> Texte important.
<sub> Texte indicé.
<sup> Texte exposant.
<u> Définit le texte qui devrait être stylistiquement différent du texte normal.
